• 1. 今天的主要工作是mongodb的集成。之前已经有安装mongodb，所以不做研究
  • 2. mongodb与node的集成框架是mongoose.可以通过npm install mongoose安装
  • 3. mongoose与mybatis，hibernate等ORM框架类似，就是对数据库连接做了一些封装，然后 一些操作语句做了封装。同时对document(类似于sql中的一行记录)和model(js对象)做了一个映射
  • 4. 一些mongodb的常用命令 show dbs:查看所有数据库 use dbname:操作某一个数据库 show collections:查看数据库中所有的集合(类似于表的概念) db.createCollection("name"):创建一个集合 db.colname.drop():删除一个集合 db.colname.insert({}):插入一个文档(类似于一行记录) 还有一些命令，在今后的使用中慢慢熟悉吧
  • 5. 获取数据库里面的一个collection之前，要先注册model
  • 6. 架构上做了一些改变。增加了一个service文件夹,专门用来存放服务器端的一些业务模块。 增加了一个db文件夹。其中包含mongodb.js用来创建数据库连接，以及一个models文件夹 用来保存一个model文件，也就是与数据库表(collection)一一对应的东西。 需要注意的是：如果我们在js里面注册了一个model：db.model('Name',{});mongoose会帮我们 在数据库中创建一个小写的且后面加了's'的集合。比如定义的model:Menu,而数据库里面会多一个 menus的集合
